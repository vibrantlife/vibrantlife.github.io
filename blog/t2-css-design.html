For the blog post I'll be describing the differences between absolute, fixed, relative, and static properties in CSS. I was stumped by this a few times, so nothing works better to understand a concept than to teach it right?

Let's start with static.


Static: This is the default of all elements. The element will be on the page where it is placed in the html. Look at the example below. Static is the last element on the html page flow. It's lives on the bottom of the page.

![Static Gif](http://g.recordit.co/pA4Ig1a7Lt.gif">Static Gif) absolute: This value allows the element to scroll with the page. Using the absolute position takes the element out of the HTML page flow. For example, if you put the element first in the HTML content, the absolute position will remove the element from the flow and it will scroll with the page. It reacts to the properties TOP, BOTTOM, LEFT, AND RIGHT. Take a look at the example below. I'm placing the element at RIGHT: 0px and position absolute.
