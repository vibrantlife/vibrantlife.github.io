Pair Programming

Brandi Ford and Nichole Shelton

What is Pair Programming

Pair programming is a structured way to tackle a problem and learn from each other.

Why is Pair Programming Important to DBC?

Pair programming is important to DBC because it allows students to learn from one another, learn through teaching, and learn different perspectives from their peers.

Compare Driver and Navigator

When someone is the driver they are doing the actual programming and action on the computer. The navigator guides and does the necessary research for the driver.

Reflection
